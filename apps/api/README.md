# @dash-finance/api

Elysia API server for Dash Finance.

## Getting Started

### Development

```bash
bun run dev
```

The API will be available at `http://localhost:3001`

### Build

```bash
bun run build
```

### Production

```bash
bun run start
```

## Project Structure

- `src/` - Source code
  - `index.ts` - Main application entry point
